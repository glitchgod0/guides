* ![Un cuadrado bronceado.](/assets/rb3pc/config/smalltan.png "Cuadrado bronceado") **Para cambiar el lenguaje del juego**:
	* **Cambia `Idioma del consola` a `Español`** para cambiar el idioma de Rock Band 3 de Ingles a Español.
	* **Cambia `Tipo de teclado` a `Teclado español`** para cambiar el teclado virtual de Ingles a Español.
